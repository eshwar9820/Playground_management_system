# Playground_management_system
The College Playground Management System is a unique 
software application, which aims to efficiently maintain the college's 
playground facilities. It streamlines the process of reserving and 
utilizing outdoor spaces on campus for everyone, including students, 
faculty, and staff. This ensures easy accessibility and good upkeep of 
the areas
Database Used: MongoDB
Development is done using
1. HTML
2. CSS
3. Bootstrap
4. Express
5. React
6. Node JS
